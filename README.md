# 프로젝트 개요
 이 프로젝트는 처음 개발을 시작하는 사람이 프로젝트의 표준 개발환경을 셋팅할 수 있고 공유할 수 있도록 돕는 프로젝트입니다. 터미널과 git의 기초 사용법을 익히고 도커를 이용해서 개발환경을 이미지화하여 어느 pc에서도 동일한 환경을 컨테이너로 재현할 수 있게 함과 동료와 표준 환경을 공유하여 사용할 수 있음을 목표로 합니다.

## 1)실행환경
- OS : macOS 15.7.4 24G517
- Shell : zsh 5.9 (x86_64-apple-darwin24.0)
- Docker : 28.5.2, build ecc6942
- Git : 2.53.0
