## Docker 컨테이너 내 터미널 접속
문제) 컨테이너의 터미널에 접속하고 싶어서 docker run -it ubuntu를 했지만 해당 컨테이너는 1회성 동작을 하고 백그라운드에서 동작하지 않아서 터미널로 접속할수 없었음.

가설) ubuntu에서 지속적인 서비스가 없기에 바로 꺼지는 것으로 가설을 세움.

해결) 대신 지속적인 서비스가 있는 이미지를 사용한 컨테이너로 변경하여 터미널 접속에 성공

## 바운트 마운트 실행 명령+호스트 변경 전/후 비교
문제) 컨테이너가 실행 중인 호스트를 변경하라는지 호스트에서 파일을 변경하라는지 정확히 파악되지 않았음.

가설) 아마도 호스트에서 파일을 변경할 경우 반영이 되는지 확인하는 것으로 가설을 세움

해결) 평가 진행을 통해서 가설과 동일하다고 판단하여 그렇게 진행함.
