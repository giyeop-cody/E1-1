# Git 초기화 및 GitHub 연동

## 1) Git 설정 레포지토리 생성 및 초기화
Git에 작성자 이름과 이메일을 설정

`git config --global user.name 사용자명`

`git config --global user.email 사용자이메일`

레포지토리 폴더 생성
`mkdir 디렉토리명`

레포지토리 폴더로 이동
`cd 디렉토리명`

레포지토리 git 초기화
`git init`
